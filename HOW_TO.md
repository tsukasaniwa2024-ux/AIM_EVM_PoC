# はじめてのセットアップガイド

このガイドを読めば、プログラムの知識がなくても **EVM 輸入帳票自動化システム** を動かせます。  
所要時間：約15〜30分

---

## 必要なもの

- Windows 10 または Windows 11 のPC
- インターネット接続
- GitHubアカウント（招待を受けた方のみ）

---

## ステップ1：Docker Desktopをインストールする

Docker Desktopは、このシステムを動かすために必要なソフトウェアです。

1. 以下のURLをブラウザで開く  
   👉 https://www.docker.com/products/docker-desktop/

2. **「Download for Windows」** をクリックしてダウンロード

3. ダウンロードしたファイル（`Docker Desktop Installer.exe`）をダブルクリックして実行

4. インストール画面の指示に従って進める（基本的にすべて「OK」「Next」で大丈夫）

5. インストール完了後、PCを**再起動**する

6. 再起動後、デスクトップまたはタスクバーに **Docker Desktop** アイコンが表示されるのでクリックして起動する

7. クジラのアイコンが表示されて、アイコンが **緑色** になれば起動完了

> ⚠️ Docker Desktopは起動に1〜2分かかることがあります。緑色になるまで待ってください。

---

## ステップ2：プロジェクトをダウンロードする

1. キーボードの **Windowsキー + R** を押して「ファイル名を指定して実行」を開く

2. `powershell` と入力して **Enter** を押す（黒い画面が開きます）

3. 黒い画面（PowerShell）に以下を**コピー&ペースト**して **Enter** を押す

```
git clone https://github.com/tsukasaniwa2024-ux/AIM_EVM_PoC.git
```

> ⚠️ `git` コマンドが使えない場合は以下の手順でGitをインストールしてください。
> 1. https://git-scm.com/download/win を開く
> 2. 「Click here to download」をクリックしてインストール
> 3. インストール後、PowerShellを開き直して上記のコマンドを実行

4. ダウンロードが完了したら、以下を入力して **Enter** を押す

```
cd AIM_EVM_PoC
```

---

## ステップ3：設定ファイルを作成する

1. PowerShellで以下を入力して **Enter** を押す

```
cp .env.example .env
```

2. 次に以下を入力してメモ帳で `.env` ファイルを開く

```
notepad .env
```

3. メモ帳が開いたら、以下のように書き換えて保存する（Ctrl+S）

```
OCR_API_KEY=ここにAPIキーを貼り付け
OCR_PROVIDER=openai
DATABASE_URL=sqlite:///./evm_import.db
```

> ⚠️ `OCR_API_KEY` は別途担当者から受け取ったキーを貼り付けてください。

---

## ステップ4：システムを起動する

1. PowerShellで以下を入力して **Enter** を押す

```
docker-compose up --build
```

2. 最初は5〜10分ほどかかります（ダウンロードが始まります）

3. 以下のような文字が表示されたら起動完了です

```
backend-1  | INFO:     Application startup complete.
```

> ⚠️ エラーが出た場合は Docker Desktop が起動しているか確認してください。

---

## ステップ5：ブラウザで開く

1. ブラウザ（Edge・Chrome等）を開く

2. アドレスバーに以下を入力して **Enter** を押す

```
http://localhost:8001
```

3. 「OCR解析・計算ツール」という画面が表示されれば成功です！

---

## 使い方

### 1. ファイルを選択する

- **PDFファイル**：インボイス（Commercial Invoice）のPDFを選択
- **画像ファイル**：パッキングリスト等の画像（JPG/PNG）を選択

### 2. 為替レートを設定する

- **帳票から自動取得**：帳票に為替レートが記載されている場合はこちらを選択
- **手動入力**：為替レートを手動で入力（1〜999の範囲）

### 3. 実行する

**「OCR解析・計算実行」** ボタンをクリック。  
数秒〜十数秒後に結果が表示されます。

### 4. 結果を確認する

- **基礎情報**：日付・インボイス番号・金額等のヘッダー情報
- **品目明細**：品番・品名・数量・金額・円換算額等の明細情報

### 5. ファイルに出力する

- **「CSVで出力」**：Excelで開けるCSVファイルとして保存
- **「Excelで出力」**：Excelファイル（.xlsx）として保存

保存場所はダイアログで自由に選択できます。

---

## システムを停止する

PowerShellで **Ctrl + C** を押すと停止できます。

次回起動時は以下のコマンドだけでOKです（`--build` は不要）。

```
docker-compose up
```

---

## 最新版に更新する

担当者から「更新があります」と連絡が来たら、以下を実行してください。

```
git pull origin develop
docker-compose up --build
```

## よくあるトラブル

### 「docker-compose up」でエラーが出る

→ Docker Desktopが起動しているか確認してください。タスクバーのクジラアイコンが緑色かどうか確認。

### ブラウザで画面が開かない

→ Docker Desktopが起動しているか、`docker-compose up` が実行中かを確認してください。

### 「処理に失敗しました」と表示される

→ `.env` ファイルの `OCR_API_KEY` が正しく設定されているか確認してください。

### 「為替レートを入力してください」と表示される

→ 為替レートを「手動入力」に切り替えて、1〜999の範囲で入力してください。
