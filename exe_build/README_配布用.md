# AIM EVM 輸入帳票自動化システム - 使い方

## 初回セットアップ（1回だけ）

1. `.env.example` を `.env` にリネームする
2. `.env` をメモ帳で開いて `OCR_API_KEY=` の後にAPIキーを入力して保存

## 起動方法

`EVM輸入帳票自動化ツール.exe` をダブルクリックするだけです。  
自動でブラウザが開きます。

## 終了方法

黒いウィンドウ（コンソール）を閉じてください。

## 配布ファイル一覧

```
EVM輸入帳票自動化ツール.exe   ← メインプログラム
.env          ← 設定ファイル（APIキーを記入）
```
