# EVM 輸入帳票自動化システム 取扱説明書

---

## 初回セットアップ（1回だけ）

### 1. `.env`ファイルの設定

同梱の `.env.example` を `.env` にリネームしてください。

**手順：**
1. `.env.example` を右クリック → 「名前の変更」
2. `.env` に変更して Enter

次に `.env` をメモ帳で開いて、APIキーを入力して保存してください。

```
OCR_API_KEY=ここに受け取ったAPIキーを貼り付け
OCR_PROVIDER=openai
DATABASE_URL=sqlite:///./evm_import.db
```

> ⚠️ APIキーは担当者から受け取ってください。

---

## 起動方法

`EVM_Import_Tool.exe` をダブルクリックしてください。

- 黒いウィンドウが開きます（閉じないでください）
- 数秒後にブラウザが自動で開きます
- 「OCR解析・計算ツール」という画面が表示されれば起動成功です

> ブラウザが自動で開かない場合は、手動で `http://localhost:8001` を開いてください。

## 使い方

### 1. ファイルを選択する

| 項目 | 内容 |
|------|------|
| PDFファイル | インボイス（Commercial Invoice）のPDF |
| 画像ファイル | パッキングリスト等の画像（JPG / PNG） |

### 2. 為替レートを設定する

| モード | 説明 |
|--------|------|
| 帳票から自動取得 | 帳票に為替レートが記載されている場合 |
| 手動入力 | 為替レートを手動で入力（1〜999） |

### 3. 実行する

**「OCR解析・計算実行」** ボタンをクリックしてください。  
数秒〜十数秒後に結果が表示されます。

### 4. 結果を確認する

| セクション | 内容 |
|-----------|------|
| 基礎情報 | 日付・インボイス番号・合計金額等 |
| 品目明細 | 品番・品名・数量・金額・円換算額等 |

### 5. ファイルに出力する

| ボタン | 出力形式 |
|--------|---------|
| CSVで出力 | CSV形式（Excelで開けます） |
| Excelで出力 | Excel形式（.xlsx） |

保存場所はダイアログで自由に選択できます。

---

## 終了方法

黒いウィンドウ（コンソール）を閉じてください。

---

## よくあるトラブル

| 症状 | 対処法 |
|------|--------|
| ブラウザが開かない | 黒いウィンドウが起動しているか確認。手動で `http://localhost:8001` を開く |
| 「処理に失敗しました」と表示される | `.env` の `OCR_API_KEY` が正しく設定されているか確認 |
| 「為替レートを入力してください」と表示される | 手動入力に切り替えて1〜999の範囲で入力 |
| 画面が真っ白 | 黒いウィンドウを閉じて再起動 |
